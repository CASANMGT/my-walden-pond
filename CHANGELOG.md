# Changelog

All notable changes to **My Walden Pond** are documented here.

Format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and versions follow [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.2.0] - 2026-07-10

### Added
- First-visit onboarding (3 screens)
- One review per day UX with view-today shortcut
- Weekly map preview on Today page
- Journal edit page and list delete
- Modern problem on chapter pages, outcome on theme pages

## [1.1.1] - 2026-07-10

### Added
- In-app About & Version page with release history
- What's new banner on home after updates
- Version panel in Settings and side menu

## [1.1.0] - 2026-07-10

### Added
- Lessons library with all 50 chapters, explanations, and life-application steps
- Entry detail page with per-entry Markdown export
- Side navigation menu with offline status
- Daily reminder notifications (Settings)
- Offline-first PWA with service worker caching
- Visual refresh aligned to design mockups (pond illustration, quote cards, theme icons)

### Changed
- Bottom navigation: Today, Lessons, Journal, Themes, Profile
- Journal entries link to full detail view with score rings

## [1.0.0] - 2026-07-10

### Added
- Initial MVP: daily lesson, review flow, localStorage journal
- Seven theme groups and weekly Walden Map
- Markdown export and data reset
- Mobile-first Next.js app deployed to Vercel
